PRINT SETUP (one-time)
1. Open Devices & Printers → right-click your P-touch → Printing preferences
2. Set paper size to match your tape width (e.g. 12mm), length = Auto
3. When printing: select your P-touch, set Margins → None, enable Background graphics
4. Match the Tape Width and Printer DPI selectors to your printer
PT-P710BT = 180dpi · PT-P950NW = 360dpi · PT-P900W = 360dpi · PT-D600 = 180dpi
