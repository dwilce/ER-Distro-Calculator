? How to Use the ER Equipment Calculator

⚡ Getting Started

Select your region (US 208V or ROW 230V) using the toggle in the top bar — all amp calculations update live. Switch between distro sheets using the dropdown selector next to the region toggle.

By default the calculator includes three standard sheets (18-Way, 48-Way, 72-Way). Opening the Distro Designer and adding your own distros creates custom sheets for your rig — the defaults restore automatically if you remove all designer nodes.

On mobile, use the bottom tab bar to switch between Distro, Inventory, and Add Equipment panels. The ☰ menu gives access to Labels, Reports, Designer, Import, Save/Load, and Help.

🔊 Assigning Equipment

  1. Click + next to any output (e.g. A1, B3) — the Add Equipment panel activates on the right
  2. Select equipment from the dropdown and set the quantity per output
  3. Choose a Sub-slot if multiple different items share one output (Master + up to 3 subs)
  4. Click Add Device & Enter Information — equipment is written to the distro sheet
  5. Fill in Unit ID, Placement, and Address for each unit
  6. Click Update & Clear — saves unit details and resets the panel

Editing: Click any populated output to reload it into the panel. Make changes and click Update & Clear to save.

Assigning to multiple outputs at once: After clicking + on any output, an Additional outputs section appears showing all ways. Click any to also assign the same equipment there — selected outputs turn navy. The button shows the total count (e.g. Add to 4 outputs). Click the section header to collapse it.

Fixed outlets (48-Way, 72-Way, PDR, Lasearray) appear below the bank grid. 1-phase fixed outlets accept equipment. 3-phase outlets link to another distro sheet — their full rated amps are reserved across all 3 phases once linked.

📄 Import from PDF or Spreadsheet

Click Import ▼ to choose a source:

PDF — extracts equipment lines from any PDF (stage plots, riders, CAD exports, equipment lists). Each extracted line is fuzzy-matched against the inventory:

  • ✓ Green — confident match. Change via dropdown if needed.
  • ? Amber — possible match. Pick from the dropdown.
  • Click ✕ to remove unwanted lines. Enter quantities in the amber box.

Tick items, choose Distro Sheet / Bank / Way / Sub-slot on the right, and click + Assign Selected. The slot advances automatically. Repeat until each item shows ✓ done, then click Done.

Spreadsheet (.xlsx or .csv) — imports a structured equipment list. Headers detected automatically: Name / Equipment / Fixture, Qty / Quantity, Unit ID / Unit No / Serial, Address / Patch / DMX, Placement / Location / Position. Pre-detected Unit IDs, addresses, and placements are written automatically on assign — no re-entry needed.

⚡ Distro Designer

Click ⚡ Designer to open the interactive power chain map — white canvas with a clean grid, Araneo-inspired node layout.

Distro types:

  • 72-Way (400A) — 12 banks A–L, 6 ways, 400A. Fixed outlets: 6×32A 1φ, 1×32A 3φ, 1×63A 3φ, 1×125A 3φ
  • 48-Way (63A) — 8 banks A–H, 6 ways, 63A/125A switchable. Fixed outlets: 3×16A 1φ, 1×32A 3φ
  • 18-Way (32A) — 3 banks A–C, 6 ways, 32A
  • SocaSwitch ER — 6-outlet Socapex breakout. Set parent distro and bank in Properties. Equipment from that bank shows as its children in the flowchart.
  • PDR — Fixed outlets only: 3×32A 1φ, 3×32A 3φ, 10×16A 1φ. Set input rating in Properties.
  • 1Ph Lasearray — 1 bank, 8 ways, 32A
  • Distribution Box, Generator / Mains, Custom Unit

Equipment cards: Each output gets a card beneath the node showing each assigned unit as its own row — equipment name, Unit ID, placement, and address. Cards are colour-coded by bank colour (matching the calculator's bank colour picker). Cards are independently draggable. Click any card to jump to that sheet.

Zones: Click ⊟ Zone to add a named coloured region. Drag to move, drag the bottom-right corner to resize. Select a zone to rename it, change colour, or delete it.

Background image: Click 🖼 BG to load a stage plot or floor plan behind the diagram. Use the BG Opacity slider in the left panel to control transparency. Remove with ✕ BG.

Connections: Click 🔗 Connect, then click source then destination. A bank-picker prompt appears if the source has multiple banks — select which bank feeds this connection. Click any connection line to select it and edit in Properties:

  • Connector type — CAM Lock, 125A, 63A CEE, 32A CEE, Socapex, Tru, Tru/DMX, Tru/Ether, PowerCon, PowerCon/DMX, PowerCon/Ether, and others
  • Cable length — 1, 2, 3, 5, 10, 20, 30, 50, 100m
  • Custom label
  • FROM — change which bank or fixed outlet this connection exits from

All auto-generated connections (equipment card lines, Socapex lines, 3-phase FO lines) are also clickable and show the same Cable Type / Length selectors. Connection colours match the bank's colour from the calculator.

Controls: Drag nodes to reposition freely. Drag equipment cards independently. ⊞ Auto Layout arranges into a top-down tree. Scroll or use +/− to zoom. ⛶ Fit All to fit everything on screen. SVG exports the diagram. PDF prints an A3 landscape version with white background.

Properties panel — rename, recolour, add a note, set SocaSwitch parent, set PDR input rating, jump to calculator sheet, or delete a node. ↺ Reset Card Positions appears if cards have been moved.

Save/Load: The designer canvas saves with .erdistro including all nodes, connections, zones, card positions, auto-connection cable data, and the background image is reloaded separately.

📊 Grand Summary

The Grand Summary bar at the bottom of each distro sheet shows total load per phase:

  • ✓ OK — Under 80% of phase limit
  • ⚠ ~84% — 80–99% — approaching capacity
  • ⚠ OVER — At or above limit

Where fixed outlets are present, the summary breaks down ways load + 1φ outlet load + 3φ reserved. For 48-Way and PDR, click the phase limit toggle to switch between 63A and 125A (or 32A and 63A for PDR).

⚠ Voltage Warnings

Items can be flagged as voltage-restricted in the Google Sheet (columns F and G). Run ⚡ ER Calculator → Add Voltage Columns once to enable.

  • 120V Only — always flagged with a 120V ONLY badge.
  • 240V Only — flagged 240V ONLY when region is set to US 208V.

Warnings appear on the distro sheet, all label types, the Equipment Report, and equipment cards in the Designer.

🖨 Printing Labels

Click Labels ▼ in the PRINT group:

  • Avery 5163 — Distro bank labels, US Letter, 10 per sheet
  • Avery L7173 — Same layout, A4
  • Equipment Labels (P-touch 24mm) — Brother PT-P710BT, one label per unit, 50mm long. 3 lines: name (bold), Unit ID · Placement, Address. Set tape width 24mm, length 50mm, margins None, background graphics on.
  • Port Labels — Standalone page for 12mm P-touch tape. Gigacore 16XT (300mm), Luminode 12 (324mm), Luminode 4 (108mm), DMX8 (200mm), LumiSplit 10 (350mm), Cable Plug (12mm × 2 per plug). Each strip shows the label length to set in the printer driver.
  • Case Labels — Opens the Flight Case Labels page (Avery 5126, 177×127mm)

🖨 Print Sheet — generates a clean A4 print of the active distro sheet with all banks, equipment, fixed outlets, and grand summary. Use browser print → Save as PDF.

📦 Flight Case Labels

Click Case Labels in the Labels dropdown. Layout: tour name band (top), description (middle), location + box number (bottom). Supports Solid, Split, Diagonal, and H-Split colour styles. Tick Add to Flight Case Labels when assigning equipment to auto-populate rows.

🖶 Equipment Report

Click Report for a printable A4 table of every assigned item across all distro sheets — distro, output, equipment, amps, voltage warnings, unit ID, placement, and address. Fixed outlet assignments included. A summary banner highlights voltage mismatches.

💾 Saving & Loading

💾 Save — Downloads your show as a .erdistro file capturing all assignments, unit details, bank colours, limits, fixed outlet data, logo, and the complete Distro Designer canvas (nodes, connections, zones, card positions, cable data).

📂 Load — Restores any previously saved .erdistro file exactly as it was, including the Designer canvas.

🔍 Inventory

The left panel lists all equipment with watts and amps. Voltage-restricted items show a badge. Search to filter. Click any item to pre-select it in the Add Equipment dropdown. To add or edit inventory, use the Google Sheet: ⚡ ER Calculator → + Add New Equipment.
