? How to Use the ER Equipment Calculator

⚡ Getting Started

Select your region (US 208V or ROW 230V) using the toggle in the top bar — all amp calculations update live. Switch between distro sheets using the dropdown selector next to the region toggle.

By default the calculator includes three standard sheets (18-Way, 48-Way, 72-Way). Opening the Distro Designer and adding your own distros replaces these with your custom rig — they restore automatically if you remove all designer distros.

On mobile, use the bottom tab bar to switch between Distro, Inventory, and Add Equipment panels. The ☰ menu gives access to Labels, Reports, Designer, Save/Load, and Help.

🔊 Assigning Equipment

  1. Click + next to any output (e.g. A1, B3) — the Add Equipment panel activates
  2. Optionally tick Add to Flight Case Labels and set the number of cases
  3. Select equipment from the dropdown and set the quantity
  4. Choose a Sub-slot for multiple items on one output (Master + up to 3 subs)
  5. Click Add Device & Enter Information — equipment is written to the distro sheet
  6. Fill in Unit ID, Placement, and Address for each unit
  7. Click Update & Clear — saves unit details, pushes to Flight Case Labels if ticked, and resets the panel

Editing: Click any populated output to reload it into the panel. Make changes and click Update & Clear to save.

Assigning to multiple outputs at once: After clicking + on any output, an Additional outputs section appears below the target showing every way in the sheet. Click any to also assign the same equipment there — selected outputs turn navy. The write button updates to show the total count (e.g. Add to 4 outputs). Click the section header to collapse it when not needed.

Fixed outlets (on 48-Way, 72-Way, PDR, and Lasearray) appear in their own section below the bank grid. Click + Assign on any 1-phase fixed outlet to assign equipment to it. 3-phase outlets link to another distro sheet via the dropdown — their full rated amps are reserved on all 3 phases once linked.

📄 Import from PDF or Spreadsheet

Click Import ▾ to choose a source:

PDF — extract equipment from any PDF (stage plots, riders, CAD drawings, equipment lists). Matched lines appear in a popup:

  • ✓ Green border — confident inventory match. The dropdown shows the matched item; change it if needed.
  • ? Amber border — possible match. Open the dropdown to pick from Best matches or All inventory.
  • Click ✕ on any row to remove unwanted lines.

Quantities must be entered manually — each row shows an amber qty box. Type the correct number; it turns green once set.

Once matches and quantities look right: tick items, choose Distro Sheet / Bank / Way / Sub-slot on the right, set quantity, and click + Assign Selected. The slot advances automatically. Repeat until each item shows ✓ done, then click Done.

Spreadsheet (.xlsx or .csv) — imports a structured equipment list. Columns are detected automatically by header name — it looks for: Name / Equipment / Fixture, Qty / Quantity, Unit ID / Unit No / Serial, Address / Patch / DMX / Channel, Placement / Location / Position. If Unit ID, Placement, or Address columns are found they show as coloured badges on each row and are written automatically when assigned — no re-entry needed. Quantities from the spreadsheet are pre-filled. The same assign panel is used as for PDF import.

⚡ Distro Designer

Click ⚡ Distro Designer to open the interactive power chain map. Use it to plan and visualise how multiple distros feed each other, and to see what equipment is assigned to each one.

Distro types available:

  • 72-Way (400A) — 12 banks, 6 ways, 400A limit. Fixed outlets: 6×32A 1φ, 1×32A 3φ, 1×63A 3φ, 1×125A 3φ
  • 48-Way (63A) — 8 banks, 6 ways, 63A/125A switchable. Fixed outlets: 3×16A 1φ, 1×32A 3φ
  • 18-Way (32A) — 3 banks, 6 ways, 32A limit
  • SocaSwitch ER — 6-outlet Socapex-fed unit. Links to a bank on a parent distro; that bank's equipment shows as its children in the flowchart
  • PDR — No bank grid. Fixed outlets: 3×32A 1φ, 3×32A 3φ, 10×16A 1φ. Select 32A or 63A input in Properties
  • 1Ph Lasearray — 1 bank, 8 ways (Powercon), 32A single-phase limit
  • Distribution Box — Custom distro with 3 banks, 6 ways
  • Generator / Mains — Power source node, no calculator sheet
  • Custom Unit — Freely labelled node

Flowchart view: Assigned equipment appears as cards beneath each distro node, colour-coded by phase. Fixed outlets appear as coloured badges between the node and the equipment cards. Click any equipment card to jump to that sheet in the calculator.

Connecting distros: Click 🔗 Connect, click the source then the destination. Choose the connector type before connecting — CAM Lock (gold), 125A (red), 63A CEE (green), 32A CEE (blue), Socapex (white). Click a connection to edit or delete it.

Controls: Drag nodes to reposition. Auto Layout arranges everything into a clean top-down tree accounting for actual node heights. +/− and scroll wheel to zoom. ⛶ Fit All fits everything to screen. Export SVG saves the diagram for documentation.

Properties panel (right side when a node is selected) — rename the distro, change colour, add a location note, set SocaSwitch parent bank, set PDR input rating, or jump to the calculator sheet. 🗑 Delete Distro removes the node and its sheet.

📊 Grand Summary

The Grand Summary bar at the bottom of each distro sheet shows total load per phase in a compact single row:

  • ✓ OK — Under 80% of the phase limit
  • ⚠ 84% — 80–99% of limit — approaching capacity
  • ⚠ OVER — At or above the phase limit

Where fixed outlets are present, the summary shows a breakdown: ways load + 1φ outlet load + 3φ reserved = total. Remaining headroom is shown in amps. For 48-Way and PDR, click the phase limit toggle to switch between 63A and 125A (or 32A and 63A for PDR).

⚠ Voltage Warnings

Items in the inventory can be flagged as voltage-restricted in the Google Sheet (columns F and G). Run ⚡ ER Calculator → Add Voltage Columns once to set this up.

  • 120V Only — always flagged with a 120V ONLY badge regardless of region.
  • 240V Only — flagged with 240V ONLY when region is set to US (208V).

Warnings appear on the distro sheet, on all label types, and in the Equipment Report.

🖨 Printing Labels

Click Labels ▼ to choose a label format:

  • Avery 5163 — Distro bank labels, US Letter, 10 per sheet (2×5), 4"×2"
  • Avery L7173 — Same layout, A4, 99.1×57mm
  • P-touch 24mm — Equipment labels for Brother PT-P710BT (or other P-touch with 24mm tape). One label per unit, max 50mm long. Shows equipment name, ID, placement, and address with voltage warning badge. Set tape width to 24mm, length to 50mm in the printer driver before printing.

Distro labels include a Fixed Outlets card after the bank labels, showing all assigned 1-phase equipment and linked 3-phase sheets. Voltage warning badges appear on all label types. Upload a logo via 🖼 Logo — it appears on all labels. Print with margins None and background graphics on.

🌐 Port Labels — Opens a standalone page for printing port identification strips on any Brother P-touch printer (12mm tape recommended). Designed for Luminex network gear and patch cables.

Device types:

  • Gigacore 16XT — 30mm box · 300mm label (10 ports)
  • Luminode 12 — 27mm box · 324mm label (12 ports)
  • Luminode 4 — 27mm box · 108mm label (4 ports)
  • DMX8 — 25mm box · 200mm label (8 ports)
  • LumiSplit 10 — 35mm box · 350mm label (10 ports)
  • Cable Plug — 12mm box (50% of EtherCon barrel), prints 2 identical labels per plug (one per end). Set quantity to number of plugs.
  • Custom — enter any pitch and port count

Each port has two text lines (main label + secondary e.g. IP/note). Multiple strips per session — all save automatically. Each strip shows a yellow badge with the exact label length to enter in the printer driver under Set Length.

Print setup: In P-touch Printing Preferences set tape width to match your tape, length = Set Length, enter the mm shown on the badge. Set Margins to None and enable Background Graphics when printing.

📦 Flight Case Labels

Click Case Labels to open the Flight Case Labels page — Avery 5126 format (177×127mm, 2 per A4 sheet).

Layout: Top third = coloured tour name band with logo. Middle = description. Bottom = location (left) and box number with total (right).

Header styles: Solid, Left/Right Split, Diagonal Stripes, or Top/Bottom Split — each supports a two-colour picker.

Auto-populate: Tick Add to Flight Case Labels when assigning equipment, fill in unit details, then click Update & Clear — rows are created automatically. Box numbers auto-increment.

🖶 Equipment Report

Click Report for a printable A4 table of every assigned item across all distro sheets — distro, output, equipment, amps, voltage warnings, unit ID, placement, and address. Fixed outlet assignments are included. A summary banner highlights any voltage mismatches at the top.

💾 Saving & Loading

💾 Save — Downloads your show as a .erdistro file capturing all assignments, unit details, region, limits, fixed outlet data, and logo.

📂 Load — Restores any previously saved .erdistro file exactly as it was.

Note: the Distro Designer canvas is not currently saved with the .erdistro file — export it as SVG for documentation before closing.

🔍 Inventory

The left panel lists all equipment with watts and amps per unit for the active region. Items flagged as 120V Only or 240V Only show a coloured badge. Search to filter. Click any item to pre-select it in the Add Equipment dropdown.

To add or edit equipment: open the Google Sheet and use ⚡ ER Calculator → + Add New Equipment. Step 4 of the form asks for voltage restriction. Run Add Voltage Columns once to enable the checkbox columns if not already present.
