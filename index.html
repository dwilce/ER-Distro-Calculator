? How to Use the ER Equipment Calculator

⚡ Getting Started

Select your region (US 208V or ROW 230V) using the toggle in the top bar — all amp calculations update live. Switch between distro sheets using the dropdown selector next to the region toggle.

By default the calculator includes three standard sheets (18-Way, 48-Way, 72-Way). If you open the Distro Designer and add your own distros, these defaults are replaced by your custom sheets — and restored automatically if you remove all designer distros.

🔊 Assigning Equipment

  1. Click + next to any output (e.g. A1, B3) — the Add Equipment panel on the right activates
  2. Optionally tick Add to Flight Case Labels and set the number of cases
  3. Select equipment from the dropdown and set the quantity
  4. Choose a Sub-slot for multiple items on one output (Master + up to 3 subs)
  5. Click Add Device & Enter Information — equipment is written to the distro sheet
  6. Fill in Unit ID, Placement, and Address for each unit
  7. Click Update & Clear — saves unit details, pushes to Flight Case Labels if ticked, and resets the panel

Editing: Click any populated output to reload it into the panel. Make changes and click Update & Clear to save.

📄 Import from PDF

Click Import PDF to extract equipment from any PDF — stage plots, riders, CAD drawings, or equipment lists. Matched lines appear in a popup:

  • ✓ Green border — confident inventory match. The dropdown shows the matched item; change it if needed.
  • ? Amber border — possible match. Open the dropdown to pick from Best matches or All inventory.
  • Click ✕ on any row to remove unwanted lines.

Quantities must be entered manually — each row shows an amber qty box. Type the correct number; it turns green once set. Quantities are never auto-detected as this proved unreliable across PDF types.

Once matches and quantities look right: tick items, choose Distro Sheet / Bank / Way / Sub-slot on the right, set quantity, and click + Assign Selected. The slot advances automatically. Repeat until each item shows ✓ done, then click Done.

⚡ Distro Designer>

Click ⚡ Distro Designer> to open the interactive power chain map. Use it to plan and visualise how multiple distros feed each other, and to see what equipment is assigned to each one.

Adding distros & creating sheets: Click a distro type in the left panel — 72-Way, 48-Way, 18-Way, SocaSwitch ER, Distro Box, Generator, or Custom. Each distro type (except Generator) automatically creates a matching calculator sheet with the correct bank, way, and phase limit configuration. The first designer distro removes the three default sheets to avoid confusion; they return automatically if you remove all designer distros.

The SocaSwitch ER is a 6-outlet unit fed via Socapex. Adding one automatically switches the active connector type to Socapex, ready to draw the feed connection. Its calculator sheet has a single bank (A) with 6 ways — one per outlet — and a 16A per-outlet limit.

Flowchart view: Once equipment is assigned in the calculator, the designer shows it as a live flowchart beneath each distro node — a trunk line drops from the distro box, a horizontal bus spans across, and individual equipment cards hang below showing the output label (e.g. A1), quantity, and equipment name. Cards are colour-coded by phase (purple Ph1, orange Ph2, teal Ph3). Click any equipment card to jump straight to that distro sheet in the calculator. The flowchart updates instantly whenever you assign or remove equipment.

Connecting distros: Click 🔗 Connect, click the source distro, then the destination. An arrow is drawn showing power flow direction. Choose the connector type from the left panel before connecting — CAM Lock (gold), 125A (red), 63A CEE (green), 32A CEE (blue), Socapex (white). Click a connection line to select it and change its type, label, or delete it.

Other controls:

  • Drag any distro node to reposition it on the canvas
  • Auto Layout — arranges nodes into a top-down power flow tree
  • Properties panel (right) — rename the distro (also renames the calculator sheet), change colour, add a location note, or jump to the calculator sheet
  • 🗑 Delete Distro — removes the node and its calculator sheet
  • 🗑 Clear All — removes all designer distros and restores default sheets
  • Export SVG — saves the full diagram as a vector file for documentation

📊 Grand Summary

The Grand Summary at the bottom of each distro sheet shows total load per phase:

  • ✓ OK — Under 80% of the phase limit
  • ⚠ 84% — 80–99% of limit — approaching capacity
  • ⚠ OVER — At or above the phase limit

For the 48-Way, click the 63A | 125A toggle in the Grand Summary to switch between phase limits.

⚠ Voltage Warnings

Items in the inventory can be flagged as voltage-restricted in the Google Sheet (columns F and G). Run ⚡ ER Calculator → Add Voltage Columns once to set this up, then tick the checkbox for any item that only works on a specific voltage.

  • 120V Only — always flagged with a 120V ONLY badge regardless of region.
  • 240V Only — flagged with 240V ONLY when region is set to US (208V).

Warnings appear on the distro sheet next to the equipment name, on all label types, and in the Equipment Report as a dedicated VOLTAGE column with a highlighted row.

🖨 Printing Labels

Click Labels ▼ to choose a label format:

  • Avery 5163 — Distro bank labels, US Letter, 10 per sheet (2×5), 4"×2". One label per bank showing all outputs, equipment, and unit IDs.
  • Avery L7173 — Same as 5163 but A4, 99.1×57mm.
  • Avery 6737 — Individual equipment labels, A4, 65 per sheet (3×13), 38.1×21.2mm. One per unit showing name, ID, placement, and address.

Voltage warnings (120V ONLY / 240V ONLY) appear on all label types for flagged items.

Upload a logo via 🖼 Logo> — it appears on all labels and the flight case page. Print with margins None, background graphics on.

📦 Flight Case Labels

Click Case Labels to open the Flight Case Labels page — Avery 5126 format (177×127mm, 2 per A4 sheet) with rounded trim corners.

Layout: Top third = coloured tour name band with logo. Middle = description. Bottom = location (left) and box number with total (right).

Header styles: Solid, Left/Right Split, Diagonal Stripes, or Top/Bottom Split — each supports a two-colour picker.

Auto-populate: Tick Add to Flight Case Labels when assigning equipment, fill in unit details, then click Update & Clear — rows are created automatically. Box numbers auto-increment.

🖶 Equipment Report

Click Report for a printable A4 table of every assigned item across all distro sheets, showing distro, output, equipment, amps, voltage warnings, unit ID, placement, and address. A summary banner highlights any voltage mismatches at the top, and a dedicated Voltage Warnings count appears in the summary row.

💾 Saving & Loading

💾 Save> — Downloads your show as a .erdistro file capturing all assignments, unit details, region, limits, and logo. Fully portable and reloadable on any device.

📂 Load> — Restores any previously saved .erdistro file exactly as it was.

Note: the Distro Designer canvas is not currently saved with the .erdistro file — export it as SVG for documentation.

🔍 Inventory

The left panel lists all equipment with watts and amps per unit for the active region. Items flagged as 120V Only or 240V Only show a coloured badge. Search to filter. Click any item to pre-select it in the Add Equipment dropdown.

To add or edit equipment: open the Google Sheet and use ⚡ ER Calculator → + Add New Equipment. Step 4 of the form asks for voltage restriction. Run Add Voltage Columns once to enable the checkbox columns if not already present.
